Quick Delete Line/Lines Plugin for Xcode

2016-04-08
  • Eclipse
  • Plugin
  • Xcode

What’s this?

I like delete line(s) feature by shortcut key (crontol/+d, command+d on Mac) so much on Eclipse, but Xcode does not has this feature.Fortunately, Xcode provides plugin mechnism, so I developed this plugin.

This plugin is so simple, and easy to use to delete line(s), the delete logic is same to Eclipse.

How to Use?

You could install it from Alcatraz, it is so easy.

Enjoy it!
