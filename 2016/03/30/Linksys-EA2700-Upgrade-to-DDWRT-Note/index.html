Linksys EA2700 Upgrade to DDWRT Note

2016-03-30
Router
DDWRT, EA2700, Router


I spent so many time to upgrade my new Linksys EA2700 router to the DD-WRT, and I get it finally.

So exciting!

Use following steps:

I’m new to dd-wrt (never used before) and linksys routers, so here are some explanation and notes i think usefull to start with dd-wrt on EA2700.

Here, this is what i did starting with the linksys smartwifi factory firmware :

  • 1) soft default reset (15s)

  • 2) flash linksys classic

    http://kb.linksys.com/Linksys/ukp.aspx?vw=1&articleid=25856

    http://downloads.linksys.com/downloads/firmware/FW_EA2700_1.0.14.129982.zip

  • 3) hard reset 30/30/30

    http://www.dd-wrt.com/wiki/index.php/Hard_reset_or_30/30/30

  • 4) flash linksys classic (even if it is already there)

  • 5) hard reset 30/30/30

  • 6) flash BrainSlayer image (build 24160)

    http://www.dd-wrt.com/phpBB2/viewtopic.php?t=261499&postdays=0&postorder=asc&start=0

    ftp://ftp.dd-wrt.com/others/eko/BrainSlayer-V24-preSP2/2014/05-27-2014-r24160/linksys-ea2700/

  • 7) After the first dd-wrt boot, set the boot options in GUI (Administration -> commands -> save startup)

    nvram set partialboots=0

    nvram commit

  • 8 ) hard reset 30/30/30

  • 9) set again the boot options wich ha been erased by the hard reset 30/30/30 (Administration -> commands -> save startup)

  • 10) Configure dd-wrt like i want

Some notes and known problems :

  • Without the falshing twice linksys classic (step 1 to 5), my EA2700 get back to stock after 3 reboots after flashing dd-wrt (step 6)

  • Without setting the boot options on step 7 and 9 my EA2700 get back to stock after reboot (seem like there will be no patch Sad http://svn.dd-wrt.com/ticket/3270)

  • 5Ghz radio is not detected, only 2.4Ghz is in the GUI (5Ghz radio is not supported but hope for usb drivers http://www.dd-wrt.com/phpBB2/viewtopic.php?p=824484#824484)

  • Check Nvram use, between 25kb and 28kb (< 32kb so i guess its ok)

    http://www.dd-wrt.com/phpBB2/viewtopic.php?p=883001#883001

    http://www.dd-wrt.com/phpBB2/viewtopic.php?p=901523#901523

  • dd-wrt shows giga build

  • QoS used for 2 days and works like a charm! Smile

Thanks BrainSlayer, Fractal, LOM , Robb and others for Serial Dump / reverse / giving us these firmwares
Thanks nolimitz and tehpensfan for the clear flash and known problems explanation for newbies and everybody at dd-wrt

Continue this good job!

Enjoy!
